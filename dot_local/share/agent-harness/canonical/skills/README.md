# Canonical Skills

Add harness-agnostic skills here using the open `SKILL.md` format.

Each immediate child directory should be a skill bundle, for example:

```text
~/.local/share/agent-harness/canonical/skills/
  my-skill/
    SKILL.md
    scripts/
    references/
```

Harness adapters project these canonical skill bundles into harness-specific skill locations.
